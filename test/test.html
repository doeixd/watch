Watch v5 - Parent-Child Component Test

This test demonstrates the new parent-child component composition features:

  • Child-to-Parent: Counter buttons expose APIs that the parent can call
  • Parent-to-Child: Child components can access parent context

Parent Component with Child Watchers

Click the counter buttons to increment. Use "Reset All" to reset all counters. Use "Show Total" to sum all counters.

Implementation Details

The implementation includes:

  • createChildWatcher() - Creates a live Map of child APIs
  • getParentContext() - Allows children to access parent context
  • runOn() - Now captures and stores generator return values
  • Generator return values are stored as public APIs
